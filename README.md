# Programy do pobrania

Publiczne repozytorium z wydaniami programu Video And Sound Downloader Pro.

Repozytorium zawiera:

- gotowe wydania programu w sekcji Releases,
- informacje o bezpieczeństwie i sumy SHA-256,
- rozszerzone snapshoty kodu źródłowego do weryfikacji transparentności,
- skrócone podglądy najważniejszej logiki,
- materiały pokazujące wygląd kolejnych wersji.

## Najnowsze wydanie

[Video And Sound Downloader Pro v8.0.8](https://github.com/Headlost/Programy-do-pobrania/releases/tag/v8.0.8) — bezpłatna wersja bez limitu czasu z pełniejszym tłumaczeniem angielskim, poprawioną diagnostyką formatów, lepszą obsługą wielojęzycznych strumieni ARTE, czytelniejszym trybem WYKRYJ oraz stabilniejszym układem okien.

Materiały transparentności v8.0.8 znajdują się w:

- `transparency/source-disclosure-v8.0.8/`,
- `transparency/source-disclosure-v8.0.8.zip`,
- `transparency/source-preview-v8.0.8.py`,
- `transparency/CHANGELOG_v8.0.8_PL.md`,
- `transparency/CHANGELOG_v8.0.8_EN.md`,
- `transparency/Aktualny.wyglad.aplikacji.do.pobierania.v8.0.8.EN.PNG`.
